# 定价方案总览

> 最后更新：2026-07-15

## 定价哲学

我们卖的不是代码，是**工程时间**和**架构确定性**。

| 分层 | 目标客户 | 定价逻辑 |
|------|---------|---------|
| 社区免费版 | 个人开发者、小团队 | 零成本获客，占领开发者心智 |
| 企业 K8s 商用版 | 中型企业、SaaS 公司 | 按年订阅，创造持续营收 |
| 源码授权版 | 大型企业、ISV | 高客单价买断，建立战略合作 |
| 架构咨询 | 决策层、CTO | 按天收费，用专业度建立信任 |

## 各产品定位

- **Form-A**：标准化产品，走量
- **Form-B**：知识产品，走质
- **分销渠道**：杠杆，放大覆盖

## 购买流程

```
社区版 → GitHub Clone / Docker Pull
企业版 → 官网下单 → License 邮件交付 → 技术支持群拉入
源码版 → 签 NDA → 预付款 → 源码交付 → 技术交接会议
架构咨询 → 预约 → 预付 50% → 交付 → 尾款
代理商 → 签约 → 生成推广码 → 客户下单自动分账
```

## 价格调整策略

- 企业版每年 1 月调价一次，涨幅 10%~20%
- 源码版按服务包弹性报价（¥198,000 起）
- 架构咨询阶梯价：5 天以上 ¥15,800/天，10 天以上 ¥12,800/天
- 代理商佣金长期有效，无阶梯
