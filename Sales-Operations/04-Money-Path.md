# 资金路径说明书 — 钱从哪里来到哪里去

## 核心问题：六个产品，六条钱路

```
产品 → 客户怎么付 → 钱到哪 → 谁分钱 → 到你手剩下多少
```

---

## 路径一：Form-A 社区版

```
价格：¥0（免费）
目的：裂变获客，不赚钱
路径：无资金流
```

---

## 路径二：Form-A 企业 K8s 版（¥49,800/年）

```
付款方式：对公转账 / 微信对公收款
   ↓
钱去哪：你的银行账户（需要一张营业执照或个体户执照）
   ↓
客户收到：授权 License 文件（通过 auth-server 生成 RSA 签名）
   ↓
License 嵌入分销商 ID → 钱全部进你账 → 月底给分销商结算 30%

路径：客户→转账到你的对公户→你确认到账→auth-server生成license发客户→完
```

**实际障碍：**
- 没有公司/个体户 → 大额对公转账收不了
- 解决方案：先走微信个人收款码，单笔≤5万。年收入高了再注册个体户

## 路径三：Form-A 源码授权版（¥198,000）

```
付款方式：对公转账 + 合同
   ↓
钱去哪：必须对公账户
   ↓
交付：全套源码 Git 仓库权限 + 授权书

路径：签合同→客户打款→你开发票→确认到账→给源码权限
```

**这条路必须注册公司或找代收。** ¥198,000 不可能走个人微信收款。

## 路径四：Form-B 架构说明书（¥2,980）

```
付款方式（选一个）：
  A. 小鹅通 / 知识星球  →  平台代收，扣6%手续费，每月结到你微信
  B. 知乎付费专栏       →  平台代收，扣5%，提现到微信
  C. 直接微信转账       →  你拉群，发PDF
   ↓
交付：PDF + 30分钟线上答疑（预约制）

路径A实际流程：
  客户在平台付款¥2,980 → 平台扣¥179 → 平台每月15号打给你¥2,801 → 你微信零钱到账
```

**推荐路径A**：小鹅通。省心，自动发货，退款纠纷平台处理。

## 路径五：Form-B 一对一咨询（¥19,800/天）

```
付款方式：微信转账 / 对公转账
   ↓
钱去哪：微信零钱 / 银行卡
   ↓
交付：线上会议/现场

路径：客户约时间→付款→远程腾讯会议或到场→完
```

## 路径六：分销商佣金结算（30%）

```
关键问题：怎么确保分销商拿到的钱和你算的账一致？

解决方案：
  auth-server 数据库中每张 license 都记录一个 distributor_id。
  你登录管理后台 → 按分销商导出结算报表 → 微信转账给分销商。
  
流程：
  1. 客户付款¥49,800
  2. 你生成 license，填 distributor_id = "agent_zhangsan"
  3. 钱¥49,800全部进你账
  4. 月底你查后台：agent_zhangsan 的活跃 license 共¥149,400
  5. 应结佣金 = ¥149,400 × 30% = ¥44,820
  6. 你微信转账¥44,820给张三
  7. 张三看到真实数据，信任建立，持续推单
```
