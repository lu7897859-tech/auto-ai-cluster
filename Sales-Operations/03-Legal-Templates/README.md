# 法务文件说明

> 本目录下的所有协议模板为**草稿版**，供内部使用和初步客户沟通。正式签署前需由公司法务审核 / 律师修订。

## 文件清单

| 文件 | 适用场景 | 状态 |
|------|---------|------|
| `community-license.md` | 社区版（GitHub 开源）许可协议 | ✅ 可用，需法务确认 |
| `enterprise-license-agreement.md` | 企业 K8s 商用版授权协议 | ✅ 模板，按客户情况修改 |
| `distributor-agreement.md` | 代理商签约协议 | ✅ 模板，可立即用于招募 |
| `form-b-purchase-terms.md` | 架构说明书 / 咨询服务购买条款 | ✅ 模板，可作为在线购买条款 |

## 使用指引

1. **社区版许可**：放在 GitHub 仓库根目录为 `LICENSE` 文件
2. **企业版授权**：每个客户单独一份，交付 License 时一并签署
3. **代理商协议**：电子签章（法大大 / 上上签 / 腾讯电子签）
4. **Form-B 条款**：架构说明书在线购买时展示为购买条款，用户勾选同意

## 注意事项

1. 所有协议中的 `[方括号]` 内容为需要填写的变量
2. 协议一式两份，双方各持一份
3. 数据安全条款需参考中国《数据安全法》《个人信息保护法》
4. 本模板不构成法律意见，建议重大合作前聘请专业律师审阅
