## 一、技术问题（1-20）

### Q1：你们需要什么样的服务器配置？

**A：** 最低 4C8G（社区版单机），推荐 8C16G。企业版建议集群 2 节点起步。硬盘 100GB+ SSD，操作系统 Ubuntu 20.04+/CentOS 7+/Debian 11+。

### Q2：我的数据放哪？

**A：** 全部存在你自己的服务器上。我们不存你的任何数据。企业版可以对接你的私有云/混合云。

### Q3：支持什么数据库？

**A：** 社区版用 SQLite（开箱即用），企业版支持 PostgreSQL 14+、MySQL 8.0+、SQL Server 2019+。

### Q4：支持国产化环境吗？（信创）

**A：** 支持。已在 KylinOS V10 + 鲲鹏/飞腾上跑过，数据库支持达梦、人大金仓。需要信创适配的走企业版。

### Q5：支持 Docker/K8s 部署吗？

**A：** 社区版提供 Docker Compose 一键部署。企业版支持 K8s Helm Chart，已在 Rancher 上验证过。

### Q6：最多能管多少个 Agent？

**A：** 社区版单机支持 50+ Agent，企业版集群支持 200+ Agent，上不封顶（看硬件）。

### Q7：Agent 之间怎么通信？

**A：** 通过内置的消息总线，支持点对点、广播、发布/订阅三种模式。企业版支持消息加密。

### Q8：你们支持什么 LLM？

**A：** 任何 OpenAI 兼容 API 的模型都能接：OpenAI、Claude、文心一言、通义千问、DeepSeek、智谱 GLM、本地部署的 ollama/vLLM 等。

### Q9：有没有 API？文档在哪？

**A：** 有完整的 REST API 和 Webhook。部署完成后访问 `http://你的地址/api/docs` 就能看到 Swagger 文档。

### Q10：工作流怎么定义？要用代码吗？

**A：** 不需要。提供 YAML 配置和可视化编辑器两种方式，拖拽式编排。想用代码也支持——YAML 写完后系统自动解析。

### Q11：支持自定义 Agent 吗？

**A：** 支持。你写一个 Python 函数，加上装饰器就注册成 Agent。Form-B 源码授权用户可以直接看到全部 Agent 实现。

### Q12：有没有日志和监控？

**A：** 有。内置操作日志、Agent 心跳监控、资源消耗看板。企业版支持对接 Prometheus + Grafana。

### Q13：离线环境能部署吗？

**A：** 能。提供离线安装包。但建议首次部署有网络，方便拉依赖。后续运行完全不需要互联网。

### Q14：支不支持 HTTPS？

**A：** 支持。推荐用 Nginx 反代 + Let's Encrypt 证书。内置也支持自签名证书用于内网环境。

### Q15：支不支持 SSO/AD/LDAP？

**A：** 企业版支持 LDAP、OAuth2（GitHub/GitLab/Google/企业微信/钉钉/飞书）。

### Q16：Agent 出错会怎么样？

**A：** 支持自动重试（可配置次数和间隔），重试仍失败则进入「异常队列」人工处理。不会丢任务。

### Q17：能不能自定义 Agent 的 Prompt？

**A：** 能。每个 Agent 可以配置独立的 System Prompt，支持模板变量和动态注入。

### Q18：有没有审计日志？

**A：** 企业版有完整的操作审计日志，谁在什么时候做了什么，全部可追溯。支持日志导出对接 SIEM。

### Q19：多租户支持吗？

**A：** 企业版支持多租户。每个租户独立 Agent 实例、独立配置、独立数据隔离。

### Q20：你们的 Agent 能联网搜索吗？

**A：** 可以。每个 Agent 可以配置工具（Tools），内置 Web Search、RAG 检索、数据库查询等工具，也可以通过 API 接入自定义工具。

**A：** 首付 50% 试用一个月，确认后付尾款。超过 15天不满意按天折算退款。

### Q28：源码授权到底包含什么？

### Q33：有没有硬件一体机方案？

**A：** 有，找我们单独询价。标准配置：2U 服务器 + 预装企业版 + 1 年技术服务。

### Q34：你们接受什么支付方式？

**A：** 对公转账（国内主推）、企业支付宝、微信支付。美元结算需额外联系。

## 三、竞品问题（36-42）

### Q36：和 LangChain / LangGraph 比怎么样？

**A：** LangChain 是一个开发框架（可以理解为 SDK），我们是一个开箱即用的产品。你拿 LangChain 要从零搭系统，我们帮你省了这一步。

### Q37：和 AutoGPT / AgentGPT 比呢？

**A：** 他们是单 Agent 工具，我们是多 Agent 集群管理。AgentGPT 启动一个 Agent 做一件事，我们管理几十上百个 Agent 协同干活。

### Q38：Dify 呢？

**A：** Dify 是 LLM 应用开发平台，偏「搭 AI 应用」。我们是「管 Agent 集群」——两个定位不同。但在工作流编排上有重叠。Dify 的 Agent 编排能力比我们弱一些，我们的可视化操作面板比 Dify 少一些。看场景选。

### Q39：为什么不开源？

**A：** 社区版免费提供全部功能，但核心源码闭源。如果你需要源码——买 Form-B。开源项目的商业模式≠我们的商业模式。

### Q40：你们有开源版 API 兼容吗？

**A：** 社区版的 API 和企业版完全兼容。从社区版升级到企业版不需要改任何一行代码。

### Q41：和微软 Copilot Studio 比呢？

**A：** Copilot Studio 是 SaaS，数据上微软云。我们是私有化部署，数据在你手里。另外 Copilot Studio ¥1,100+/月/用户，我们企业版 ¥49,800 买断不限用户。

### Q42：和百度的 AppBuilder 比呢？

**A：** AppBuilder 是百度云上的 PaaS 服务，技术栈深度绑定百度。我们是平台无关的私有化部署。

## 四、售后问题（43-50）

### Q43：出问题了怎么办？

**A：** 社区版：GitHub Issues / 微信群求助。企业版：5×8 电话/微信响应，2小时内响应，紧急情况 1小时。

### Q44：技术支持包含哪些内容？

**A：** 部署支持、配置指导、故障排查、Bug 修复。不包含：定制开发、你代码的 Debug、Agent Prompt 优化。

### Q45：你们提供定制化开发吗？

**A：** 提供，按项目评估报价。标准费率 ¥2,000/人天，最低 5 人天起。

### Q46：我想让你们的工程师帮我写工作流，多少钱？

**A：** 首次免费（1 个工作流）。之后按复杂度报价，一般在 ¥500~¥3,000/个工作流。

### Q47：买了之后多久能部署好？

**A：** 社区版 15 分钟（如果网络好）。企业版首次部署产品方可远程协助，一般在 1 个工作日内完成。

### Q48：版本多久更新一次？

**A：** 大约每 2~3 个月一个正式版本。紧急 Bug 修复不受版本周期限制。

### Q49：我买了企业版，你们如果倒闭了怎么办？

**A：** 
> 实话：我们目前没有倒闭风险，但如果你担心——

> 3. 社区版/Form-B 用户完全不受影响

### Q50：你们有 SLA 吗？

**A：** 企业版 SLA：系统可用性 99.9%（按年计算），单次故障修复时间 ≤ 4 小时。社区版无 SLA。
