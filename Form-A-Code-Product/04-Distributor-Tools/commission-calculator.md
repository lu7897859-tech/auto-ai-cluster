## 佣金结算规则

| 规则 | 说明 |
|------|------|
| **结算周期** | 月度结算，次月15日前到账 |
| **起付门槛** | 单笔佣金 ≥ ¥500 即结算，低于 ¥500 累积到下月 |
| **退款处理** | 客户退款时，对应佣金从下月佣金中扣除 |
| **续费佣金** | 客户续费年度技术服务费，代理商同样享受 30% 佣金 |
| **开票** | 佣金可开服务费发票（增值税普通发票） |
| **税率** | 个人代理按劳务报酬代扣代缴；企业代理对公转账不开票则无扣除 |

## 快速口算公式

> **佣金 = 客户付款金额 × 0.3**
>
> 举例：客户付了 ¥50,000，佣金就是 ¥15,000

**简易换算表：**
