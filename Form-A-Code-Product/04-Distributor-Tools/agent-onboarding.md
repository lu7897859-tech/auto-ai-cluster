# 📋 代理商开通流程

> 从申请到接单，6 步走完。

---

## 第1步：填写申请表

扫码或点击链接填写代理商申请表：

> **[申请表链接](#)** ← 找产品方获取实际链接

需要填写的信息：
- 姓名 / 公司名称
- 手机号 / 微信号
- 所在城市
- 主要行业方向（IT/制造/金融/教育/政务/其他）
- 预期月销量
- 技术能力（有/无技术团队）——这决定了你适合推什么版本

> 提交后 **24小时内** 会收到审核结果通知（微信/短信）。

---

## 第2步：获取分销商身份

审核通过后，你会收到：

```
您的分销商ID：agent_[名字拼音]_[三位数字]
示例：agent_zhangsan_001
```

**分销商ID的作用：**
- 客户注册时填写此 ID，系统自动关联到你名下
- 佣金计算以此为准
- 每个分销商 ID 绑定一个微信/手机号

**重要：** 分销商 ID 请妥善保存，所有推广物料中需包含此 ID。

---

## 第3步：获取演示环境

你会收到以下信息：

| 项目 | 说明 |
|------|------|
| 演示地址 | `https://demo.yourcompany.com/agent_[id]`（示例，实际地址另行通知） |
| 管理员账号 | `admin_[id]@demo.com` |
| 初始密码 | 首次登录后强制修改 |
| 演示 License | 90 天有效期，到期前联系续期 |
| 部署脚本 | `deploy.sh` 和 `deploy_offline.sh`（离线版本） |

**建议收到后：**
1. ✅ 马上登录一次，确认能通
2. ✅ 修改密码
3. ✅ 跑一遍 `deploy.sh`，确认你也能部署
4. ✅ 按 `demo-checklist.md` 逐个过一遍

---

## 第4步：加入代理商社群

扫码加入代理商微信群：

- **群名：** [Auto-AI-Cluster 代理商之家]
- **群功能：**
  - 产品更新通知（重要）
  - 报备客户（防止撞单）
  - 技术答疑（产品方技术人员在线）
  - 成功案例分享
  - 佣金到账通知

**入群后第一步：**
> 修改群昵称为：`[分销商ID]-姓名-城市`

---

## 第5步：参加线上培训

> 时长约 30 分钟，每周二/四下午各一场，支持回看。

**培训内容：**

| 模块 | 时长 | 内容 |
|------|------|------|
| 产品概述 | 5min | Auto-AI-Cluster 解决什么问题、三个版本区别 |
| 演示流程 | 10min | 标准演示流程 + 常见翻车点 |
| 销售话术 | 5min | 三个场景的话术核心 |
| 佣金政策 | 5min | 佣金计算 + 结算流程 |
| Q&A | 5min | 现场答疑 |

**培训结束后你会拿到：**
- 1 份培训证书（电子版）
- 全套销售物料（PDF 版）
- 1 个可对外展示的 Demo 账号

> 建议新代理商在 **成为分销商后 7 天内** 完成培训。

---

## 第6步：开始接单

完成以上步骤后，你就可以开始接单了。

**标准接单流程：**

```
  你接触客户 → 确认需求 → 约演示
       ↓
  你完成演示（或拉产品方协助）
       ↓
  客户有意向 → 你在群内报备客户信息
       ↓
  产品方出正式报价/合同
       ↓
  客户签约 → 产品方交付 → 你拿到佣金
```

**⚠️ 撞单处理规则：**

| 场景 | 处理 |
|------|------|
| 你在微信群报备过的客户，别人签了 | 归你 |
| 你未报备的客户，别人先报备了 | 归先报备者 |
| 客户自己找到产品方，没提你 | 没佣金 |
| 你和客户已进入报价阶段，另一个代理商介入 | 产品方协调，保护先入者 |

**核心规则：** 有客户先报备。发一句「客户报备：XXX公司，联系人XXX」到群里，就算你的。

---

## 常见问题

### Q：我推客户，但我不懂技术，演示怎么办？

**A：** 你只管约时间拉群，产品方技术人员远程演示。你主要做商务侧的推动。演示成功后佣金按 30% 照算。

### Q：我可以同时做多个产品的代理吗？

**A：** 可以。但若推广竞品建议提前告知，我们来判断是否冲突。

### Q：我推荐了客户但客户没马上买，以后客户自己找你们买了，算我的吗？

**A：** **算。** 你在系统里报备过的客户，6 个月内签约都算你的。

### Q：我在社交媒体上推广，二维码用什么？

**A：** 培训结束后你会拿到带分销商 ID 的专属推广二维码和素材包。

### Q：我可以发展下级代理吗？

**A：** 目前不支持多级分销体系。但你可以推荐朋友也来做代理商——他出单你拿 5% 的推荐奖金。
