Key Architecture Decisions

  • Docker Compose over K8s: K8s control plane needs 2C2G before workload; Docker Compose + healthchecks achieves 95% reliability at 10% resource cost
  • Patroni over cloud RDS: Managed PostgreSQL -50/mo; Patroni on same VPS adds 128MB overhead
  • RSA-4096 over OAuth: Offline capability critical for edge deployments without internet
  • Circuit-breaker AI gateway: Eliminates single-provider dependency and cost volatility
